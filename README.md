# Log Stride
## Get Started
- Clone And Install Packages
```
# Clone Repo
git clone git@github.com:zalosspace/logStride.git
# Install Packages
npm install
```
- Setup `.env` file
```
VITE_SUPABASE_URL=https://YOUR_PROJECT_ID.supabase.co
VITE_SUPABASE_PUBLISHABLE_DEFAULT_KEY=YOUR_PUBLISHABLE_KEY
```
- Run
```
# http://localhost:6969
npm run dev
```
